# Terjemahan-Bima-Indonesia
Model dibangun menggunakan OpenNMT dg data latih 10.000 diperoleh nilai Bilingual Evaluation Understudy (BLEU) tertinggi 77,20, skor Translation Edit Rate (TER) terendah 23,10 dan skor Metric for Evaluation of Translation with Explicit ORdering (METEOR) 0,78 sehingga model yang dihasilkan masuk dalam kategori baik.
